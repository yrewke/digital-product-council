# Commercial Scope

Accept a direction only when all four tests hold:

1. Restaurant value is meaningful.
2. Fulfilment works economically and operationally.
3. The offer competes against real alternatives, including doing nothing.
4. The provider can repeatedly sell, deliver, support, and profit.

Treat implementation as a black box unless it materially changes customer value, restaurant burden, provider cost, support, scalability, reliability, legal/commercial risk, profitability, or time to sell.
