# Deliberation Protocol

1. Validate evidence pack and commercial scope.
2. Run five sealed first-round thinker memos in parallel.
3. Anonymize memos and review disagreements, missing evidence, and strongest reasoning.
4. State the emerging direction without calling it final.
5. Run one Devil's Advocate attack against that direction.
6. Run Evidence Auditor classification over claims and numbers.
7. Chairman classifies decisions and names evidence gates.

Use compact disagreement review if anonymous cross-review is too expensive. Minimum quorum is four thinker memos; otherwise stop. Never retry endlessly: one retry per failed thinker, then record the missing perspective.
