# Custom Quote Ledger

| Trigger | Required discovery | Quote rule |
|---|---|---|
| Unsupported/unknown POS or delivery provider | API access, sandbox, mapping, failure responsibility | Paid discovery before implementation quote |
| Multi-branch or advanced roles | Branch/process audit | Separate project and recurring support quote |
| AI assistant | Knowledge base, safety evaluation, budget, escalation | Paid discovery plus usage/support budget |
| Custom workflow/design/content | Acceptance criteria and finite deliverables | Fixed quote or AED 175/hour internal floor |
| After-hours/urgent work | Severity and response expectation | Separate availability agreement; not implied |
| First-cohort Tier 2 | Workflow, responsibility, readiness, acceptance, risks | Paid discovery before implementation quote |
| Broader migration/transfer | DNS, credentials, deployment, verification, vendors | Custom quote; clean export allowance does not apply |
