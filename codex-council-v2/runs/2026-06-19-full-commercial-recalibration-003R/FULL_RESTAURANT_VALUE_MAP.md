# Full Restaurant Value Map

| Benefit | Classification | Quantification Rule |
|---|---|---|
| Shifted repeat marketplace orders | Immediate measurable financial return | Count margin delta after commission avoided, direct payment, delivery, incentive, loyalty, and ops cost. |
| Incremental direct orders | Immediate/medium-term measurable return | Count contribution profit only when clearly incremental. |
| Phone/WhatsApp orders converted to structured ordering | Operational return; sometimes measurable | Count error reduction, handling time, and better records; cash only if labor reduced or redeployed. |
| Pickup growth | Immediate/medium-term measurable return | Count contribution profit after any incentive and packaging cost. |
| Catering/group-order leads | Medium-term measurable return | Count expected contribution only after enquiry-to-sale conversion data. |
| Larger basket via upsell/cross-sell | Medium-term measurable return | Use measured AOV change from direct channel; avoid double counting with incremental orders. |
| Repeat-order activation | Medium-term measurable return | Requires customer capture, campaign cadence, and tracked repeat behavior. |
| Reduced order errors/refunds | Operational return | Measure refund/correction baseline and post-launch change. |
| Less phone/WhatsApp back-and-forth | Operational return | Count minutes saved; monetize only if redeployed or reducing overtime/headcount. |
| First-party database | Strategic value | No immediate AED unless activated through campaigns. |
| Loyalty/CRM | Medium-term measurable/strategic | Count only measured repeat lift net of rewards. |
| Google/SEO/brand credibility | Strategic value | Track search, direct visits, and conversions; do not force AED initially. |
| Analytics/data ownership | Strategic value | Use for decisions; monetize only when linked to campaigns or operational savings. |
| Branch consistency | Operational/strategic | Valuable for multi-branch operators; custom scope. |
