# Advisor Memo: Restaurant Operator / Fulfilment Manager

**Commercial question:** Under which fulfilment model does owned direct ordering create positive ROI versus alternatives?
**Verdict:** conditional
**Strongest finding:** Outsourced delivery and hybrid fulfilment are the safest operational bridges when delivery density is uneven.
**Evidence used:** Run 001 evidence on delivery failures, integration workflows, and support burden; operational assumptions in Run 002A.
**Critical assumption:** Staff can maintain menu, status, and handoff discipline during normal and peak periods.
**Tradeoff:** Pickup-only is operationally clean, but it may undercut perceived value if the customer wants delivery.
**Disagreement expected:** The owner economist will favor pickup as the baseline; I think hybrid is more realistic once delivery exists.
**Evidence request:** On-time rate, failed-delivery rate, and staff minutes per order by fulfilment type.
